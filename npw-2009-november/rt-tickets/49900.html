If the third argument is omitted the string's length in characters is
inserted in its place.

$ ../../parrot perl6.pbc /var/src/v6/v6-KindaPerl6/t/kp6/23-str-substr.t
1..2
ok 1
ok 2

The patch also adds tests to t/01-sanity/07-substr.t

.sub 'substr'
.param string x
.param int start
- .param int len
+ .param int len :optional
+ .param int has_len :opt_flag
+ .local pmc s
+
+ if has_len goto end
+ s = new 'Perl6Str'
+ s = x
+ len = s.'chars'()
+
+ end:
$S0 = substr x, start, len
.return ($S0)
.end
