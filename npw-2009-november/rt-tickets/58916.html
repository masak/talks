Hello.

There is yet-another-lexpads-handling-bug in rakudo/parrot.

Sample program:

my @reftypes = ();
sub foo($a) {
push @reftypes, $a;
$a;
};

sub bar($s) {
$s('a');
}

sub baz {
foo('c');
}

bar(&foo);
say +@reftypes; # output 0 instead of 1
foo('b');
say +@reftypes; # output 1 as expected after first "failure"
bar(&foo);
say +@reftypes; # output 2!!!
