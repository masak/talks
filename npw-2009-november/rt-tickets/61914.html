This works fine:

class Foo {
has ($.this,
$.that,
);
}
my Foo $foo .= new( this => 3, that => 4 );
say $foo.this;


However, any whitespace after the opening paren causes a parse failure:

class Foo {
has ( $.this,
$.that,
);
}
my Foo $foo .= new( this => 3, that => 4 );
say $foo.this;
