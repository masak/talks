If the rule is called correctly it appears to work fine:

$ cat x
grammar Foo {
token foo { 'foo' }
};

grammar Bar is Foo {
token bar { <foo> | 'bar' }
};

if 'foo' ~~ /<Bar::bar>/ { 'matched'.say; }

$ ./parrot perl6.pbc x
matched
$
