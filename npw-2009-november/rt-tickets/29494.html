Hi,
I wrote a small module that uses Test::More and allows us to write
many many tests fast; For each test, it just checks whether a particular
string matches (or doesn't match) a particular rule, without paying
attention to what it parsed as. Sometimes this is all that is needed. It
is possible to mark things "TODO" or "SKIP". Here is part of the test the
patch contains to illustrate what I mean:

linetests directive => 1, <<'END';
package
module
TODO class
TODO role
END

linetests binary => 0, <<'END';
0b03
END

linetests sigparam => 1, <<'END';
$a
$a is rw = 9
+$a
END

So far I only have a few tests in there, but I'll add many more soon.

--abhijit

The current plan for tests for perl6 on parrot is to run the official
perl6 tests, which currently means "whatever is in pugs". If you're
interested in committing tests to pugs, commit bits there are freely
available.

There is also a new implementation of perl6 on parrot. See
