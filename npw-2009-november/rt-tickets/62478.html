$ perl6 -e ' for (my $i = 1; $i <= 3; $i++) { say $i; }'
2

$ perl6 -e ' for (my $i = 1; $i <= 3; $i++) { say $i; say $_ }'
2
1

So the "++" is applied before the loop body but after the assignment to $_
