Upon further reading of S03, it turns out that it isn't done quite
right. .contains isn't really a membership test, it's a test of
subsethood... which means it doesn't do what it's supposed to right now,
when sent two lists, for example:

@a.contains(@b) # should return Bool::True iff all(@b) eqv any(@a)

.namespace ['Perl6Hash']
.sub 'contains' :method
- .param pmc key
- $I0 = exists self[key]
+ .param pmc key :slurpy
+ .local pmc it
+
+ it = iter key
+ $I0 = 0
+ loop:
+ unless it goto done
+ $P0 = shift it
+ $I0 = exists self[$P0]
+ unless $I0 goto loop
+
+ done:
.return( $I0 )
.end
