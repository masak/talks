And thus it can be worked around, for the time being, doing this:

my $string = '%CD';
while $string ~~ /\%(..)/ {
$string ~~ /\%(..)/
my $match = $0;
}

...or by using binding, as moritz++ discovered:

my $string = '%CD';
my $M := $string ~~ /\%(..)/;
while $M {
my $match = $M[0];
}
